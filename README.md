# Analisi-delle-criticit-del-sistema-sanitario-italiano
Analisi delle criticità del sistema sanitario italiano

Questo progetto di data analysis è finalizzato all’identificazione e alla valutazione delle criticità del sistema sanitario italiano a livello regionale.

Attraverso la costruzione di indicatori sintetici e l’integrazione di diverse variabili (pressione ospedaliera, capacità e utilizzo delle risorse), il progetto consente di individuare le aree più vulnerabili e supportare la pianificazione strategica degli interventi sanitari.

Obiettivi dell’analisi

Costruire un indicatore di criticità sanitaria regionale

Analizzare le differenze territoriali nel sistema sanitario

Identificare le regioni con maggiore rischio operativo

Supportare la definizione di priorità di intervento

Fornire strumenti per il decision making basato sui dati.

KPI (Key Performance Indicators)

Il progetto integra diversi indicatori per valutare la criticità:

Indice di pressione ospedaliera (%)

Tasso di utilizzo dei posti letto (%)

Capacità del sistema sanitario (posti letto)

Indice sintetico di criticità (%).

Metodologia

L’approccio adottato prevede:

Costruzione di indicatori compositi

Normalizzazione delle variabili

Analisi comparativa tra regioni

Ranking territoriale delle criticità

Identificazione di pattern di rischio

Analisi dei quadranti.

Uno degli elementi centrali del progetto è l’analisi della relazione tra:

Capacità ospedaliera (posti letto)

Pressione del sistema sanitario

Il grafico a dispersione è suddiviso in quattro quadranti:

Alto a destra (↑ pressione, ↑ capacità)
→ aree ad alta criticità e possibile sovraccarico strutturale

Alto a sinistra (↑ pressione, ↓ capacità)
→ criticità legata a carenza di risorse

Basso a destra (↓ pressione, ↑ capacità)
→ situazioni efficienti con margine operativo

Basso a sinistra (↓ pressione, ↓ capacità)
→ aree a bassa domanda o sotto-utilizzo.

Dataset

I dati utilizzati derivano da fonti open data ufficiali e includono:

Attività ospedaliera

Capacità delle strutture sanitarie

Indicatori di utilizzo dei servizi.

I dataset sono stati integrati e armonizzati per consentire analisi comparative.

Data Preparation

Pulizia e standardizzazione dei dati

Gestione valori mancanti e outlier

Costruzione di variabili derivate

Integrazione multi-fonte

Creazione modello dati per Power BI.

Dashboard Power BI

La dashboard sviluppata consente di:

Visualizzare il livello di criticità per regione

Analizzare la distribuzione geografica delle criticità

Esplorare le relazioni tra capacità e pressione

Identificare rapidamente le aree prioritarie.

Insight principali

Dall’analisi emergono:

Presenza di forti squilibri territoriali

Alcune regioni presentano elevata criticità sistemica

La criticità è spesso legata alla combinazione di:

alta pressione

elevato utilizzo delle risorse

Evidenza di aree con margini di miglioramento operativo

Necessità di interventi mirati per ottimizzare la distribuzione delle risorse.

Implicazioni

Il progetto permette di:

Supportare la pianificazione sanitaria regionale

Identificare le priorità di intervento

Migliorare l’allocazione delle risorse

Monitorare il rischio di sovraccarico del sistema sanitario.

Strumenti utilizzati

Power BI – sviluppo dashboard

DAX – costruzione indicatori e KPI

Data modeling – gestione relazioni tra dataset.
